# PixelStrip
PixelStrip is a fully client-side image editor that runs entirely in the browser with no backend, no installation, and no build step required. It supports background removal powered by Transformers.js running locally via WebAssembly, along with a full suite of painting tools including pencil, brush, eraser, shapes, text, and flood fill. Images can be resized, cropped with aspect ratio control, flipped, rotated, and exported as PNG, JPG, or ICO. The interface features resizable panels, mobile support with a bottom sheet and scrollable toolbar, a magnifying glass zoom tool, and a dark industrial yellow theme.
